# 🎓 Career Guide / AI Counsellor System

A simple Career Guidance System developed using **C++ and HTML**.  
The system helps students choose suitable career paths by asking questions related to interests, aptitude, and academic background.

This project simulates the process of a **career counsellor**, guiding students after class 10 or 12 to explore possible career options.

---

## 📌 Project Overview

Choosing the right career path can be confusing for students.  
This project provides a basic **career counselling system** that asks the user questions and suggests possible streams or careers.

The system combines:
- **C++** for logic and decision making
- **HTML** for displaying the interface

---

## ✨ Features

- Ask students questions about:
  - Interests
  - Subjects they enjoy
  - Future goals
- Suggest career streams such as:
  - Engineering
  - Medical
  - Commerce
  - Arts
- Simple interactive counselling flow
- Beginner-friendly implementation

---

## 🛠 Technologies Used

- **C++**
- **HTML**
- Object-Oriented Programming Concepts
- Conditional Logic
